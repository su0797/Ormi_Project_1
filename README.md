# 사용자를 위한 음악추천 서비스 
- openAI API를 이용하여 사용자가 선택한 조건에 맞는 노래를 추천한다. 

## 1. 목표와 기능
### 1.1 목표
- HTML, CSS, JavaScript로 프로젝트를 완성한다.
- 사용자가 선택한 조건에 맞는 값을 답변으로 출력한다. 
- index 화면과 main 화면을 분리하여 구현하고 페이지간의 이동이 가능하도록 만든다.

### 1.2 기능
- 사용자가 원하는 조건의 음악 추천
